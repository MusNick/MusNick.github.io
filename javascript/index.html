# 斐波拉契数列

缓存每次已经计算过的值

function f(n: number): any  {
    let cache: any = {};
    function cacheFn(n: number): any {
        if(n === 1 || n === 2) {
          cache[n] = 1
        }
        if(!cache[n]) {
          cache[n] = cacheFn(n - 1) + cacheFn(n-2)
        }
        return cache[n]
    }
    return cacheFn(n)
}

# utils缓存函数

缓存纯函数 减少每次调用函数计算

public cached(fn: any){
    let cached = Object.create(null);
    return (function cachedFn(str: string) {
      const hit = cached[str]
      return hit || (cached[str] = fn(str))
    })
}

# iframe和浏览器回退

  1. 使用iframe，并且iframe会更新src的时候，iframe会和父窗口共用history，从而导致浏览器后退点击2次才回退

解决方案：改变src的时候 动态插入iframe（相当于每次建立新的iframe，不会产生history）

  1. 禁止浏览器后退 无刷新，体验效果好
parent.addEventListener('popstate', function(event) {
    history.pushState(null, null, document.URL);
});
