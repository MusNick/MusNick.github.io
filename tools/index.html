# 终端录屏软件

安利一款终端录屏软件terminalizer，生成gif动图(上传图片的工具是iPic)

  1. 安装terminalizer

npm install -g terminalizer

  1. 开始录屏

terminalizer record demo

  1. CTRL+D 停止录屏 生成.yaml文件

  2. 生成gif图片

terminalizer render demo

  1. 重新查看录制内容

terminalizer play demo
