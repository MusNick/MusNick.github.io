    # loaderOptions

    预处理器 loader 传递选项, 传入共享的全局变量

    WARNING

    慎用该方式注入全局变量，因为每一个模块都会将改样式打包一份，大量重复代码

    module.exports = {
  css: {
    loaderOptions: {
      // 给 sass-loader 传递选项
      sass: {
        data: `@import "@/variables.scss";`
      }
    }
  }
}
