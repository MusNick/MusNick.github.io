# git hooks

场景

🎉 代码提交前自动语法格式化并再次检查

借助husky

cnpm install husky --save

这次案列使用pre-commit, 并且适用了eslint的自动修改

(--fix自动修改, --ext 后缀名，前面记得加目录参数)

"husky": {
  "hooks": {
    "pre-commit": "eslint --fix . --ext .js"
  }
}

旧的实现方式

  1. 安装lint-staged
  2. package添加钩子
"gitHooks": {
  "pre-commit": "lint-staged"
},
"lint-staged": {
  "*.js": [
    "vue-cli-service lint",
    "git add"
  ],
  "*.vue": [
    "eslint --fix",
    "vue-cli-service lint",
    "git add"
  ]
}
